# Freelance Portfolio — Upwork Job Radar

Interactive freelance portfolio built with React + Vite. Features a real-time Upwork job radar that displays live scraping results from the [upwork-scraper](https://github.com/damarff/upwork-scraper) project.

## Features

- **Live Job Radar** — auto-refreshing feed of Upwork jobs across 20+ keywords
- **Profile Section** — skills, stats, social links
- **Featured Systems** — portfolio of shipped automation projects
- **Particle Field** — interactive background animation
- **Aurora Theme** — animated gradient mesh background

## Tech Stack

- React 19 + Vite
- Supabase (job data backend)
- Lucide icons
- CSS animations (aurora, particle field, orbs)

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
